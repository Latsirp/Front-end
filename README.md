## **Pelo Começo**

Se você gostaria de ter uma cópia local desse projeto funcionado, siga essas etapas.

### **Pré-requisitos**

#### Para front-end:

- **Visual Studio Code**
  - ESLint
  - Live Server

#### Para back-end:

- **Eclipse**
- **Intellij IDEA**
- **NetBeans**

#### Para database:

- **MySQL**

### **Instalação**

#### Front-end

1. **Abra um terminal no Visual Studio Code:**

   ```sh
   Ctrl + '
   ```

   ou

   ```sh
   Ver > Terminal
   ```

2. **Clone o repositório:**

   ```sh
   git clone https://github.com/Latsirp/Estatium.git
   ```

3. **Abra a pasta:**

   ```sh
   Ctrl + K + O
   ```

   ou

   ```sh
   Arquivo > Abrir Pasta...
   ```
